# community

Community-contributed [YakYak](https://yakyak.ai/) campaigns and channel showcases.

Each subfolder is a self-contained example channel — an exported campaign and the
prompts/settings behind it — that creators have shared so others can learn from,
fork, or remix it. YakYak supports this directly: a campaign can be exported as
JSON (`workflow/export-campaign`) and imported by someone else
(`workflow/import-campaign`) or forked (`workflow/fork-campaign`).

| Channel | Description |
|---------|-------------|
| [`BreakingBricksNews/`](BreakingBricksNews/) | A satirical brick-built news channel. |
| [`TulipHunt/`](TulipHunt/) | A serialized adventure/treasure-hunt channel. |

## Contributing

Add a folder named after your channel containing the exported campaign JSON and a
short `README.md` describing the style, premise, and any notable settings. See the
existing channels for the layout.
