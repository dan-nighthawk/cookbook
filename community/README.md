# community

This `/community` section is a collection of third-party examples supplied by our
users — community-contributed [YakYak](https://yakyak.ai/) campaigns and channel
showcases. **Please submit a GitHub Pull Request if you'd like to show off your
project here.**

Each subfolder is a self-contained example channel — an exported campaign and the
prompts/settings behind it — that creators have shared so others can learn from,
fork, or remix it. YakYak supports this directly: a campaign can be exported as
JSON (`workflow/export-campaign`) and imported by someone else
(`workflow/import-campaign`) or forked (`workflow/fork-campaign`).

| Channel | Description |
|---------|-------------|
| [`TulipHunt/`](TulipHunt/) | Self-guided motorcycle roadbook tours, turned into an episodic video channel. |

## Contributing

Want to show off your project? **Open a GitHub Pull Request.** Add a folder named
after your channel containing the exported campaign JSON and a short `README.md`
describing the style, premise, and any notable settings. See the existing channels
for the layout.
